## 1. 功能与体验
- [x] 1.1 公共查询无登录、无 D1 持久化，使用 Worker 代理外部接口。
- [x] 1.2 公共页面样式对齐私有仪表盘，自动预填 localStorage 的公共 Key，并在存在保存值时自动查询。
- [x] 1.3 支持多条查询记录列表，展示掩码 Key/额度/时间窗/使用率；重复 Key 覆盖原行，不新增重复；单行刷新/删除可用。

## 2. 安全与隔离
- [x] 2.1 localStorage 仅存公共 Key 列表与结果；不写入 D1 或其他服务器存储；日志不暴露完整 Key。
- [x] 2.2 公共查询缓存与私有 ServerState/自动刷新隔离，私有登录与数据管理行为保持不变。

## 3. 验证
- [x] 3.1 未登录：连续输入不同 Key 追加多行，重复 Key 覆盖原行，刷新/删除单行正常；自动查询可用。
- [x] 3.2 重载页面：localStorage 列表恢复，单行刷新/删除有效；无 Key 时提示输入。
- [x] 3.3 登录后：私有功能正常，公共查询不写入 D1；公共列表与私有数据互不影响。
